#Terminal based Constellations management system

##Available Options

0. Exit;
1. Add constellations (name, number of stars and type(v-visible/i-invisible));
2. Sort constellations descending by number of stars and display them in terminal;
3. Display the visible constellations with a higher number of stars than a specified value from the keyboard;
4. Create a vector with the constellation's names length and display them in terminal.
